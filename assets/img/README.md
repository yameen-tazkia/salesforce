# Image assets

Place brand and image assets in this folder.

## Logo

The site looks for **`logo.png`** here:

```
assets/img/logo.png
```

Recommended specs:
- PNG with transparent background (or matching dark navy)
- ~600px wide minimum (~1200px ideal for retina displays)
- Roughly square or slightly wider than tall

If the file is missing, the nav and footer fall back to a gradient "T" mark
plus the "Tazkia Intelligence" wordmark — so the site never looks broken.

## How to add

Either commit the file directly via Git, or use the GitHub web UI:

1. Open https://github.com/yameen-tazkia/salesforce/tree/main/assets/img
2. Click **Add file → Upload files**
3. Drag your `logo.png` in and commit.
